# EverShop Demo - Manual Testing & Bug Reporting Project

## Project Overview

This repository contains a manual testing project conducted on the EverShop Demo E-commerce application.

The objective of this project was to perform exploratory testing, identify defects, document bugs, and provide supporting evidence through screenshots and video recordings.

Application Under Test (AUT):
https://demo.evershop.io/

---

## Project Scope

The following modules were tested:

* User Registration
* Login & Authentication
* Password Reset
* Input Validation
* Product Listing
* Add to Cart
* Checkout Process
* Responsive Design
* User Interface

---
